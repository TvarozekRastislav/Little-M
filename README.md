## Princíp  
Porojekt je zložený z dvoch častí: 
- ### zariadenie na dát
- ### vozidlo

### Zariadenie na zber dát
Skladá sa z USB Adaptéra pre ESP-01, ESP8266 ESP-01 a tlačidla ktoré je používané na sprístupnenie programovancieho módu.\
**súčiastky**:
- Adaptér pre ESP-01
- ESP8266 ESP-01
- tlačidlo
